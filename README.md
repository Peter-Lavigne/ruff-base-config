# ruff-base-config

Base config for the ruff linter that I copy and extend across multiple repos.

This is a workaround until either I automate it or https://github.com/astral-sh/ruff/issues/12352 is implemented.

## Usage

Copy the `copied` directory into your repo and extend the config in `copied/pyproject.toml`.
